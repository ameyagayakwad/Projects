# Stock Market Price Prediction Using Machine Learning

## Objective
Predict future stock prices based on historical data using various machine learning models.

## Tools & Libraries
- Python, Pandas, NumPy, Matplotlib, Seaborn
- Scikit-learn, yfinance
- Optional: TensorFlow/Keras for LSTM

## Models Used
- Linear Regression
- Random Forest Regressor
- XGBoost Regressor

## Dataset
Historical data fetched using `yfinance` (e.g., Apple Inc. - AAPL)
